Projects

Granblue API 2020-2022

An API that provides equipment and character data about the mobile video game 'Granblue Fantasy'. There are many different frontend applications that continuously re-source the data for their own ends. What this project aims to solve is free and accessible data to create more third-party tools. It also serves as a way for me to modularize data and make my own frontend to showcase data for my own community within the game.

DDR Leaderboard Web App October 2022

A website that displays a leaderboard sourced from data stored on Google Sheets. Made in 1 week! Hosted with App Engine using Google Cloud.

Discord Bot 2018-2020

This Discord bot was made to make displaying course descriptions and shuttle bus times in a cleaner and accessible format to students. It used Cobalt API, an API made by other UoFT students to fetch the data.

DDR Scores Tracker 2019-2020

A mobile app that tracks scores on the go when playing the commercial arcade game 'Dance Dance Revolution'. I've been meaning to have a portable way of keeping track of my scores, and the sentiment goes to a bunch of others who play it. There are a couple of apps for this out already but I also wanted to have an AI function to it since the break time between songs is very short, inputting info needs to be done fast and simple. Frontend is using Xamarin UI framework. On the backend, AWS is used to image analyze photos and Realm as an offline database.

Filename Scrambler 2019

Bash script that alphanumerically scrambles the names of files in a folder. This was used to manage my background image names by making them anonymous.

Personal Website 2017-2022

This site your reading right now! I've been maintaining this personal website throughout the years. At first, it was created with only HTML and CSS. It's evolved to use React as a front-end framework to maintain modular components for reuse in other web projects and/or extensions to this website.
